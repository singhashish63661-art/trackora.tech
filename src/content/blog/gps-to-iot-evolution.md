## GPS: A Necessary but Incomplete Foundation

Every modern fleet starts with GPS. Knowing where your vehicles are is table stakes. But location data alone answers only one question: *where?*

The real questions fleet operators need to answer are:

- **Why** did that driver brake hard at kilometer 47?
- **When** will that engine need servicing before it fails on the highway?
- **How much** fuel was lost to idling at the Mumbai depot last week?

These questions require IoT.

## The IoT Fleet Stack

A fully instrumented Trackora deployment layers data from multiple sensor types:

### Layer 1: GPS & Cellular (Foundation)
- Real-time location with 5-second refresh
- Geo-fencing with instant alerts
- Trip history and route replay

### Layer 2: OBD-II & CAN Bus (Diagnostics)
- Engine RPM, coolant temperature, fault codes
- Predictive maintenance alerts (e.g. brake pad wear estimation)
- Fuel consumption per trip

### Layer 3: IoT Sensors (Operational Intelligence)
- Fuel rod sensors for independent fuel level monitoring
- Temperature sensors for refrigerated cargo
- Door open/close events for load security

### Layer 4: ADAS Video Telematics (Safety)
- Forward collision warning
- Lane departure detection
- Driver fatigue monitoring
- Incident footage storage
